== Description ==

Tweet Muncher is a WordPress plugin that searches for a search term using the Twitter Search API and imports the results as posts hourly.

== Installation ==

1. Upload the tweet-muncher directory to your `/wp-content/plugins/` directory
2. Activate the plugin in the 'Plugins' menu in WordPress
